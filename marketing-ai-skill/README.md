# Marketing AI Skill — Claude Code

Tres comandos de marketing digital con IA para Claude Code.

## Instalación

```bash
cd ~/Downloads
unzip marketing-ai-skill.zip
bash marketing-ai-skill/install.sh
```

## Comandos

| Comando | Descripción |
|---------|-------------|
| `/market-seo <url>` | Análisis SEO técnico y on-page completo |
| `/market-audit <url>` | Auditoría completa de marketing digital |
| `/market-competitors <url>` | Análisis competitivo con benchmarking |

## Ejemplos

```
/market-seo https://www.holaluz.com
/market-audit https://www.holaluz.com
/market-competitors https://www.holaluz.com
```

## Qué incluye cada comando

### `/market-seo`
- SEO on-page: title, meta, headings, keywords, imágenes, links
- SEO técnico: velocidad, mobile, HTTPS, schema, canonical
- Puntuación global 0-100
- Plan de acción con 10 acciones priorizadas por impacto

### `/market-audit`
- Propuesta de valor y posicionamiento
- UX y conversión (CRO): CTAs, funnel, prueba social
- Estrategia de contenido y canales digitales
- Hoja de ruta a 90 días

### `/market-competitors`
- Identificación de 5 competidores directos + 3 indirectos
- Tabla comparativa multidimensional
- Mapa de posicionamiento y gaps de mercado
- Estrategia competitiva recomendada

## Requisitos

- [Claude Code](https://claude.ai/code) instalado
- Funciona con cualquier modelo Claude (Sonnet, Opus, Haiku)
